# Cromwell CMS

Cromwell CMS is WordPress-like CMS for Next.js websites

Cromwell CMS [official site](https://cromwellcms.com/) and [github](https://github.com/CromwellCMS)

## Start

```sh
docker-compose up -d
```

## Stop

```sh
docker-compose stop
```

## Uninstall

```sh
// after stop
docker-compose rm
```
